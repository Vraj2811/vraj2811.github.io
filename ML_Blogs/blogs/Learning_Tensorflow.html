Comparing Linear Model Framework and Manual Implementation

\[ f_{\mathbf{w},b}(x^{(i)}) = \mathbf{w}\cdot x^{(i)} + b \]

prediction_tf = model(X_train)
prediction_np = np.dot(X_train, set_w) + set_b

plt.subplot(1, 2, 1)

\[ f_{\mathbf{w},b}(x^{(i)}) = g(\mathbf{w}x^{(i)} + b) \]
